## 🔍 关键概念索引

### 数据层
- **Financial Datasets API**: 主要数据源
- **缓存机制**: 内存缓存，自动去重
- **Pydantic模型**: 数据验证和类型安全

### 智能体层
- **18个智能体**: 不同投资策略
- **LLM集成**: 使用LangChain调用各种LLM
- **信号生成**: buy/sell/hold/short + 置信度

### 工作流层
- **LangGraph**: 状态机工作流框架
- **并行执行**: 多个智能体同时工作
- **状态合并**: 自动合并节点输出

### 回测层
- **时间序列模拟**: 逐日执行交易
- **交易执行**: 买入/卖出/做空/平仓
- **性能指标**: Sharpe、Sortino、最大回撤
